# Минимальный анализатор языка на Flex и Bison

Проект содержит простой лексический и синтаксический анализатор для мини-языка.

## Поддерживаемые конструкции
- Объявление переменных: `int x;`, `int y = 5;`
- Присваивание: `x = y + 1;`
- Арифметические выражения: `+`, `-`, `*`, `/`
- Условие `if`
- Цикл `while`
- Блоки `{ ... }`

## Токены лексера
- `IDENTIFIER` — идентификаторы
- `NUMBER` — целые числа
- `KEYWORDS` — `int`, `if`, `while`
- `OPERATORS` — `=`, `+`, `-`, `*`, `/`, `==`, `!=`, `<`, `>`, `<=`, `>=`
- `DELIMITERS` — `(`, `)`, `{`, `}`, `;`

## Файлы
- `lexer.l` — правила Flex
- `parser.y` — грамматика Bison (БНФ-подобная)
- `input.txt` — входной файл с исходным кодом
- `Makefile` — сборка и запуск

## Сборка
Убедитесь, что установлены `flex`, `bison`, `gcc`.

```bash
make
```

## Запуск
```bash
make run
```

Анализатор читает входные данные **только** из файла `input.txt`.

## Прямой запуск без Makefile
```bash
bison -d -Wall parser.y
flex lexer.l
gcc -Wall -Wextra -std=c11 parser.tab.c lex.yy.c -o analyzer
./analyzer
```

## Диагностика ошибок
- Лексическая ошибка: выводится сообщение с символом и номером строки.
- Синтаксическая ошибка: выводится сообщение с номером строки.
